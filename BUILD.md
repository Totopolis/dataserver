# dataserver
Open source data warehouse around sql server mdf structure

2015.

------------------
Build instructions
------------------

Required tools:
* Cmake 
* Git
* c/c++ compiler (gcc or visual studio or clang)

-----------------
Linux compilation
-----------------

$ git clone https://github.com/Totopolis/dataserver.git
$ cd dataserver
$ mkdir build
$ cd build
$ cmake ..
$ make
